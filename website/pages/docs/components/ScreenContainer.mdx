# ScreenContainer

A screen wrapper to constraint page layout.

<carbon-ad />

## Usage

`ScreenContainer` constraints content of the page using `max-width` and `padding`.

```js live noInline
import { ScreenContainer } from 'smooth-asciidoc/components'

render(<ScreenContainer>Container</ScreenContainer>)
```
